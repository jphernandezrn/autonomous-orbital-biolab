# Autonomous Orbital BioLab (BOLAB-X1)

**Nurse-led autonomous CubeSat platform for in-space antimicrobial drug-response testing and biological research in low Earth orbit.**

Stellar Life Support Technologies LLC is developing a free-flying, software-defined orbital wet-lab infrastructure designed to overcome the limitations of crew-dependent and high-cost orbital bioscience missions.

## Project Overview

**Built for modularity. Designed for scale.**

The **BOLAB-X1** is a scalable, autonomous CubeSat BioLab that enables repeatable, crew-free biological experimentation in microgravity. This nurse-led initiative aims to provide reliable, high-quality biological data for translational healthcare, pharmaceutical research, and space medicine.

## Key Features

- Free-flying autonomous orbital wet-lab platform
- Sealed microfluidics with timed dosing
- Environmental sensing and autonomous execution
- Onboard data logging and downlink capability
- Modular and scalable architecture
- Designed for antimicrobial drug-response testing and microbial adaptation studies

## Vision

To build a practical, cost-effective, and scalable orbital bioscience infrastructure that accelerates scientific discovery and supports translational health applications.
